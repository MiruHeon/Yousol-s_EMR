# Yousol's EMR (의무전자기록)
종이 기반의 환자 진료 기록을 디지털화하여 전산 시스템에 저장하고 관리하는 체계인 EMR(의무전자기록)을
Winform을 이용하여 구현하였습니다.
이를 통해 의사가 아니더라도 간접적으로 환자 진료 기록 작성을 체험해볼 수 있습니다.

### 주요 기능
간편한 소견서 작성기능<br>
대기환자 목록 시각화 (DB 사용 X)

### 기술적 특징
Winform과 C#을 이용하여 개발되었습니다.

### 느낀점
이렇게 간접적으로 환자의 진료 기록을 직접 작성해보면서, 진료기록 정보 보호가 중요하다는 사실을 깨달았고 앞으로는
더욱 더 보안을 갖춘 EMR 시스템을 구현해보고 싶다는 생각이 들었습니다.

### 주의 사항
※ 본 소프트웨어는 의료용이 아니며, 실제 진료 기록 관리용으로 사용될 수 없습니다.
